```

dirp,  and  returns it in the caller-allocated buffer pointed to by enâ€
try.  For details of the dirent structure, see readdir(3).

A pointer to the returned buffer is placed in *result; if  the  end  of

It is recommended that applications use  readdir(3)  instead  of  readâ€
```
